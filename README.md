GearGuard is a web-based **equipment tracking system** designed to monitor and manage equipment details efficiently.  
The project mainly focuses on a **clean, modern UI** and **basic equipment tracking functionality** using modern web technologies.

---

##  Features

- Equipment listing and tracking
- View equipment details
- Aesthetic and responsive UI using Tailwind CSS
- Fast frontend performance with Next.js
- MySQL database used for storing equipment data

> **Note:**  
> - Login / Signup functionality is **not implemented**  
> - Admin module is **not implemented** in the current version

---
